
# Prime Bunker — Website

Next.js 14 (App Router) + Tailwind CSS. Day/Night (Trading) theme via `next-themes`.  
Routes:
- `/` Marine Fuel Supply & Bunker Trading
- `/port-coverage`
- `/fuel-procurement`
- `/operations`
- `/compliance`
- `/market-intelligence`
- `/sustainability`
- `/risk-advisory`
- `/about`
- `/contact`

## Dev
```bash
pnpm i    # or npm i / yarn
pnpm dev  # http://localhost:3000
```

## Deploy
Push to GitHub → Vercel builds automatically.  
Connect GoDaddy DNS: A @ -> 76.76.21.21, CNAME www -> cname.vercel-dns.com

## Images
Replace `/public/og-day.jpg` and `/public/og-night.jpg` with your day/night hero images.
